Command line UI is in file P1.java.

To run the program follow the instructions below (this is for MacOs):

1. Place all the files in a folder
2. Navigate to the directory in which the folder is located
3. Compile all .java files using javac
4. Execute p1_create.sql in the docker environment
5. Run the program by running the following command "java -cp .:db2jcc4.jar P1 ./db.properties"
